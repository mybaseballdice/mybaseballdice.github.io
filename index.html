My Baseball Dice Game

Written by Caesar A.

This is a simple little baseball dice game I made up a while ago back in 2021 when I was bored. It uses a single six-sided die and two special result lists to simulate batting outcomes. The way the game works is that players roll the die and reference the first result list, if a "Hit" (or an H) gets rolled then they have to roll again and reference the second result list.

First Result List

  • 1 - Hit (H)
  • 2 - Hit (H)
  • 3 - Strikeout (K)
  • 4 - Pop Out (PO)
  • 5 - Fly Out (FO)
  • 6 - Ground Out (GO)

Second Result List

  • 1 - Single (1B)
  • 2 - Double (2B)
  • 3 - Triple (3B)
  • 4 - Home Run (HR)
  • 5 - Fly Out (FO)
  • 6 - Single (1B)

Because this game uses a highly simplified version of the standard baseball rules, only things such as base runner positions, outs, scored runs, and innings must be kept track of by the player. Things such as strikes, balls, errors, and other stuff are completely ignored. Below are some statisics, which were calculated from the afformentioned result lists.

In Game Statistics

  • Hits - 1.67
  • Outs - 4.33
  • At-Bats - 6
  • Batting Average - .278

I like this game quite a lot because the rules are easy to remember and it is very fun to play.
